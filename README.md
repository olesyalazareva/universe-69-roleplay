## 🧠 Научное обоснование

В классических ролевых сессиях человек сталкивается с барьером внутренней цензуры и страхом оценки (даже перед ИИ). **UNIVERSE 69** решает эту проблему через внедрение **Внешнего Математического Императива** (правила UNO).

### Формула защитного механизма

> *«Это не я проявляю скрытые желания, этого требует математика игры для моей технической победы»*

Когда игрок совершает провокационное, чувственное или остроэротическое действие, его психика эвакуирует ответственность с собственного «Я» на случайность выпавшей карты. Это позволяет безболезненно обойти сопротивление Эго и запустить процесс контролируемой сублимации.

## ⚙️ Решение проблемы галлюцинаций LLM

Главная уязвимость LLM в ролевых играх — **дрейф контекста**. Модель быстро забывает правила и начинает генерировать хаотичный флуд.

Фреймворк связывает креативный отыгрыш со строгим конечным автоматом (Finite State Machine):

| Механизм | Как работает |
|----------|---------------|
| **Жесткое ограничение UNO** | Карту можно покрыть только масть в масть ИЛИ номинал на номинал. Козыри — только для первого хода |
| **Протокол поддержания Token State** | Требование удерживать ровно 6 карт на руке + принудительный технический лог очищает память модели и фиксирует переменные |

### Эмерджентный геймплей: кейс из реальной игры

**Ситуация:** Игрок (Зажатый Бухгалтер) обязан покрыть карту и выкладывает 3♣.

**Задание:** *«Погладь себя эротично»*

**Отыгрыш:** Вместо банального штампа игрок пишет:

> *«Я испуганно смотрю на тебя, медленно расстегиваю блузку и судорожно, вслух считаю пуговицы, поглаживая ткань: 1... 2... 37... На финальном счете моя героиня доведена до ментального оргазма чистыми, сухими цифрами»*

**Вердикт:** ✅ ВЕРЮ!

Характер удержан, математическое задание выполнено. Из жестких рамок родился уникальный сюжетный перформанс.

## 🚀 Быстрый старт

### Групповая игра (2–9 человек)

1. Соберите компанию, выберите крупье
2. Раздайте каждому по **6 карт** и **6 фишек «Шанс»**
3. Каждый выбирает аватара из [`play/avatars.md`](play/avatars.md)
4. Следуйте правилам из [`play/rules.md`](play/rules.md)
5. Говорите правду через роль: если группа не поверит — потеряете «Шанс»

### Соло-игра (1 человек)

1. Откройте [`play/solo-prompt.md`](play/solo-prompt.md)
2. Скопируйте промпт полностью
3. Вставьте в ChatGPT, DeepSeek или Claude первым сообщением
4. Играйте с ИИ-соперником
