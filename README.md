# UNIVERSE 69

> *Ролевая игра с AI-ведущим. Исследуй персонажей, сценарии и сюжетные развилки. Соло-режим с ИИ-компаньоном или групповые сессии (2–9 игроков). 54 сюжетных триггера, система аватаров и динамический движок повествования.
---

## Описание

**UNIVERSE 69** — ролевая система с AI-ведущим для проживания фантазий и теневых идентичностей. Участник принимает роль (аватар) и проживает сценарии, недоступные в обычной жизни. Игровой цикл построен на принципах проективной психологии: внешний триггер (карта-задание) провоцирует спонтанный отклик, обходящий социальные фильтры и защитные механизмы Эго. Верификация BIVREST («Верю / Не верю») фиксирует конгруэнтность отыгрыша, делая процесс наблюдаемым и структурированным.

**Формат:** живая сессия (2–9 участников) или соло-режим с ИИ-ведущим.

**Ключевые элементы:**
* 52 карты-триггера, распределённые по уровням глубины
* 2 джокера, меняющие правила игры
* Реестр аватаров: социальные и теневые роли (18+)
* Экономика токенов «Шанс»
* 9 базовых фантазийных психологических сценариев

---

## Ключевые концепции

* **LLM Context Framing** — глубокая адаптация модели через психоаналитические паттерны
* **Cognitive Engineering** — использование психомеханик для погружения в отыгрыш
* **High-Retention Architecture** — механики вовлечения и удержания внимания
* **BIVREST Protocol** — бинарная верификация конгруэнтности («Верю / Не верю»)

## Быстрый старт

### Соло с AI
1. Открой `play/solo-prompt.md`
2. Скопируй промпт целиком
3. Отправь первым сообщением в ChatGPT / Claude / DeepSeek
4. Выбери аватар
5. Тяни карту
6. Отыгрывай

### Групповая сессия
1. Собери 2–9 участников
2. Раздай аватары
3. Перемешай колоду
4. Запусти игровой цикл
5. Фиксируй конгруэнтность через BIVREST

### В цифровом продукте
1. Интегрируй логику из `play/rules.md`
2. Подключи базу карт из `play/cards.md`
3. Настрой сбор BIVREST-логов по спецификации `play/logs.md`
