# UNIVERSE 69 — AI Companion for Your Fantasies

> **The engine utilizes applied psychoanalysis for deep contextual adaptation of LLMs. The system establishes a strict contextual corridor (roleplay sandbox), enabling the model to consistently maintain its roleplay framing and generate Mature (18+) content without scenario breaks or hallucinations. It’s not a bug — it’s architecture.**

## Overview

**UNIVERSE 69** is a roleplay system with an AI Game Master designed for exploring fantasies and shadow identities. Participants step into a role (avatar) and navigate scenarios unavailable in everyday life. The gameplay loop is built on projective psychology principles: an external trigger (a prompt card) elicits a spontaneous response, bypassing social filters and Ego defense mechanisms. BIVREST verification (“I believe / I don’t believe”) logs roleplay congruency, making the process structured and observable.

**Format:** Live session (2–9 players) or solo mode with an AI Game Master.

**Key Elements:**
* 52 trigger cards distributed across depth levels
* 2 jokers that alter the game rules
* Avatar registry: social and shadow roles (18+)
* "Chance" token economy
* 9 core fantasy erotic scenarios

## Quick Start

### Solo with AI
1. Open `play/solo-prompt.md`
2. Copy the entire prompt
3. Send as the first message to ChatGPT / Claude / DeepSeek
4. Choose an avatar
5. Draw a card
6. Begin roleplaying

### Group Session
1. Gather 2–9 participants
2. Distribute avatars
3. Shuffle the deck
4. Start the gameplay loop
5. Track congruency via BIVREST

### Digital Product Integration
1. Integrate the logic from `play/rules.md`
2. Connect the card database from `play/cards.md`
3. Set up BIVREST log collection per `play/logs.md`
